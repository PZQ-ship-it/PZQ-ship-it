Selected Research 科研经历

CogPace-Bench: A Benchmark for Cognitive Pacing and Graph-based Reasoning in LLM Educational Agents

First Author | Advisor: Haobo Wang

Feb 2026 — Present

  • Research Overview: Developed the first dynamic benchmark combining knowledge graph distance and cognitively-restricted virtual student agents to address the expert blind spot and cognitive overload in LLM tutoring.
  • Graph-based Reasoning Path Tracing: Built a dynamic sandbox driven by local knowledge graphs; engineered an interceptor engine to map entities to graphs and proposed the Graph Semantic Jump Distance (GSJD) metric to detect far-jump logical gaps and quantify the coherence of multi-step pedagogical reasoning.
  • Dynamic Adaptation and Pedagogical Stealth Quantification: Simulated student cognitive overload to evaluate dynamic pacing and epistemic humility; introduced a Mechanical Awareness Penalty (MAP) via LLM-as-a-judge to measure pedagogical stealth and learning flow maintenance.
  • Outcomes: Formulated GSJD and MAP and constructed a complete dynamic sandbox evaluation framework, laying the foundation for an upcoming academic publication or open-source release.

Digital Textbook: Core Algorithm Research for End-to-End Intelligent Authoring Platforms

Core Algorithm Researcher | Advisor: Haobo Wang

Dec 2025 — Present

  • Overview: Constructed a full-link generation and review architecture integrating GraphRAG and neuro-symbolic engines to address logical drift and attention decay in long-form professional textbook creation.
  • Multi-hop Reasoning via GraphRAG: Built a heterogeneous knowledge graph using Milvus and implemented BFS topology-aware recall to inject long-term memory, improving cross-chapter logical consistency by 38.4%.
  • Dynamic Context Pruning and Alignment Control: Designed a dual-layer pipeline and graph-distance-based context pruning to remove noise and improve outline alignment; integrated style feature injection (HNSW) to match domain-specific writing styles.

Team Leader & Core Model Researcher | Advisor: Haobo Wang
